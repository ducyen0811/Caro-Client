Cài package:

npm install
Cấu hình

Tạo file .env ở thư mục gốc nếu project của bạn đang dùng biến môi trường.

Ví dụ:

REACT_APP_API_URL=http://localhost:4000
REACT_APP_SOCKET_URL=http://localhost:4000

Nếu project hiện tại đang hard-code localhost:4000 trong code thì có thể bỏ qua bước này.

Chạy project
npm start
